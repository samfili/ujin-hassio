# Changelog

Все важные изменения в этом проекте будут документированы в этом файле.

## [1.0.0] - 2026-01-05

### Добавлено
- ✅ Полная интеграция с Ujin API
- ✅ Email-аутентификация с кодом подтверждения
- ✅ Платформа switch для всех устройств
- ✅ Поддержка Ujin Connect-din (dinrelay_m4)
- ✅ Поддержка Ujin Connect-dim (ujin-zdm-m2)
- ✅ Поддержка Ujin Aqua (ujin-zld-m1)
- ✅ Автоматическое обновление состояния каждые 30 секунд
- ✅ Хранение токена между перезагрузками
- ✅ Обработка истечения токена
- ✅ Полная документация API
- ✅ Примеры автоматизаций

### Технические детали
- Email-аутентификация с двухфакторным кодом
- Автоматическое извлечение area_guid из API
- Корректный парсинг структуры устройств
- Правильная отправка команд управления
- Все устройства отображаются как switch entities
