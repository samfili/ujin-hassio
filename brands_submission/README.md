# Ujin Smart Home - Brands Submission

Файлы для добавления иконки интеграции в официальный brands repository Home Assistant.

## Файлы

- `ujin/icon.png` (256×256) - основная иконка
- `ujin/icon@2x.png` (512×512) - высокоразрешающаяся версия

## Как добавить в brands repository

1. Форкните репозиторий https://github.com/home-assistant/brands
2. Создайте папку `custom_integrations/ujin/`
3. Скопируйте файлы icon.png и icon@2x.png в эту папку
4. Создайте Pull Request
